## Estructura del Proyecto

- `Subscriber.java` - Clase que representa un suscriptor de StreamFlix
- `SubscriptionPlan.java` - Clase que representa un plan de suscripción
- `PlaybackSession.java` - Clase que relaciona suscriptor y contenido

- `Main.java` - Punto de entrada del programa

## Ejecución

```bash
javac *.java
java Main
```
